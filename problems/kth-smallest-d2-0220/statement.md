# K 番目に小さい値

難易度: 2 / 4 ・ トピック: array

N 個の整数と整数 K が与えられる。昇順に並べたとき K 番目に小さい値を出力せよ (1-indexed)。

## 制約

- 1 <= K <= N <= 50
- 各要素の絶対値 <= 100

## 入力

```
N K
a_1 ... a_N
```

## 出力

K 番目に小さい値を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
1 1
5
```

出力:

```
5
```


### 例 2

入力:

```
3 2
3 1 2
```

出力:

```
2
```
