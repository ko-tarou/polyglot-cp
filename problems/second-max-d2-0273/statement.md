# 2 番目に大きい値

難易度: 2 / 4 ・ トピック: array

N 個の整数のうち、2 番目に大きい (異なる) 値を出力せよ。 異なる値が 2 種類未満なら `-1` を出力する。

## 制約

- 1 <= N <= 50
- 各要素の絶対値 <= 100

## 入力

```
N
a_1 ... a_N
```

## 出力

2 番目に大きい値、または `-1` を出力する。


## 入出力例


### 例 1

入力:

```
1
5
```

出力:

```
-1
```


### 例 2

入力:

```
3
2 2 2
```

出力:

```
-1
```
