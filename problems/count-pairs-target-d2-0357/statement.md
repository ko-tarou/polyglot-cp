# 和が K になる組の数

難易度: 2 / 4 ・ トピック: search

N 個の整数と目標値 K が与えられる。和が K になる相異なる添字の組 (i<j) の個数を出力せよ。

## 制約

- 1 <= N <= 200
- 各要素, K の絶対値 <= 10^6

## 入力

```
N K
a_1 ... a_N
```

## 出力

組の個数を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
4 4
1 3 2 2
```

出力:

```
2
```


### 例 2

入力:

```
3 100
1 2 3
```

出力:

```
0
```
