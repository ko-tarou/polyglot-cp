# 線形探索 (最初の位置)

難易度: 1 / 4 ・ トピック: search

N 個の整数と目標値 X が与えられる。X が最初に現れる位置 (1-indexed) を出力せよ。存在しなければ `-1` を出力する。

## 制約

- 1 <= N <= 8
- 各要素, X の絶対値 <= 20

## 入力

```
N X
a_1 ... a_N
```

## 出力

位置、または `-1` を出力する。


## 入出力例


### 例 1

入力:

```
3 2
1 2 3
```

出力:

```
2
```


### 例 2

入力:

```
3 9
1 2 3
```

出力:

```
-1
```
