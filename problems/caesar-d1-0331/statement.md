# シーザー暗号

難易度: 1 / 4 ・ トピック: string

英小文字からなる文字列 S と整数 K が与えられる。各文字をアルファベット上で K だけ後ろにずらして (z の次は a に循環) 出力せよ。

## 制約

- 1 <= |S| <= 10
- 0 <= K <= 25

## 入力

```
S
K
```

## 出力

変換後の文字列を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
abc
1
```

出力:

```
bcd
```


### 例 2

入力:

```
xyz
3
```

出力:

```
abc
```
