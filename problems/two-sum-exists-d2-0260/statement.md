# 2 数の和が K になるか

難易度: 2 / 4 ・ トピック: array

N 個の整数と目標値 K が与えられる。異なる 2 要素の和が K になる組が存在すれば `Yes`、なければ `No` を出力せよ。

## 制約

- 2 <= N <= 200
- 各要素, K の絶対値 <= 10^9

## 入力

```
N K
a_1 ... a_N
```

## 出力

`Yes` または `No` を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
2 3
1 2
```

出力:

```
Yes
```


### 例 2

入力:

```
2 100
1 2
```

出力:

```
No
```
