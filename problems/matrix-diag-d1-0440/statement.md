# 正方行列の対角成分の和

難易度: 1 / 4 ・ トピック: matrix

N x N の整数行列が与えられる。主対角成分の和を出力せよ。

## 制約

- 1 <= N <= 10
- 各要素の絶対値 <= 10^6

## 入力

```
N
行列 (N 行 N 列)
```

## 出力

対角成分の和を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
2
1 2
3 4
```

出力:

```
5
```


### 例 2

入力:

```
6
-601 451 791 850 607 -667
514 -242 136 320 261 -280
800 583 -785 555 776 -788
-217 -307 -433 93 -967 94
-560 -59 -680 -729 -118 -254
935 -728 177 599 -731 178
```

出力:

```
-1475
```
