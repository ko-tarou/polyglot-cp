# 行列の転置

難易度: 1 / 4 ・ トピック: matrix

R 行 C 列の整数行列が与えられる。転置 (C 行 R 列) を出力せよ。

## 制約

- 1 <= R, C <= 6
- 各要素の絶対値 <= 10^6

## 入力

```
R C
行列 (R 行 C 列)
```

## 出力

転置行列を R 列 C 行で出力する。


## 入出力例


### 例 1

入力:

```
2 3
1 2 3
4 5 6
```

出力:

```
1 4
2 5
3 6
```


### 例 2

入力:

```
5 3
-206 534 -575
450 -854 -341
721 -864 744
-834 -614 845
119 324 -750
```

出力:

```
-206 450 721 -834 119
534 -854 -864 -614 324
-575 -341 744 845 -750
```
