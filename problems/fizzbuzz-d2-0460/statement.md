# FizzBuzz

難易度: 2 / 4 ・ トピック: simulation

整数 N が与えられる。1 から N まで各行に出力する。ただし 3 の倍数は `Fizz`、5 の倍数は `Buzz`、両方の倍数は `FizzBuzz` と出力せよ。

## 制約

- 1 <= N <= 10000

## 入力

```
N
```

## 出力

N 行に渡って出力する。


## 入出力例


### 例 1

入力:

```
1
```

出力:

```
1
```


### 例 2

入力:

```
15
```

出力:

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
```
