# 正方行列の対角成分の和

難易度: 1 / 4 ・ トピック: matrix

N x N の整数行列が与えられる。主対角成分の和を出力せよ。

## 制約

- 1 <= N <= 10
- 各要素の絶対値 <= 10^6

## 入力

```
N
行列 (N 行 N 列)
```

## 出力

対角成分の和を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
2
1 2
3 4
```

出力:

```
5
```


### 例 2

入力:

```
7

108 -771 -229 318 -111 -250 264

-574 -407 987 -222 -591 -181 978
-762 358 -43 -548 -939 -425 -621
```

出力:

```
-1368
```
