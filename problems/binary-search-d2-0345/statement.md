# 二分探索 (存在判定)

難易度: 2 / 4 ・ トピック: search

昇順にソート済みの N 個の整数と目標値 X が与えられる。X が含まれていれば `Yes`、なければ `No` を出力せよ。

## 制約

- 1 <= N <= 200
- 各要素, X の絶対値 <= 10^9

## 入力

```
N X
a_1 ... a_N (昇順)
```

## 出力

`Yes` または `No` を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
3 2
1 2 3
```

出力:

```
Yes
```


### 例 2

入力:

```
3 5
1 2 3
```

出力:

```
No
```
