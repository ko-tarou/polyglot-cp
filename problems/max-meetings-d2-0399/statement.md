# 区間スケジューリング

難易度: 2 / 4 ・ トピック: greedy

N 個の区間 (開始 s_i, 終了 e_i) が与えられる。互いに重ならないように選べる区間の最大数を出力せよ (端点の一致は重ならないとみなす)。

## 制約

- 1 <= N <= 200
- 0 <= s_i < e_i <= 10^9

## 入力

```
N
s_1 e_1
...
s_N e_N
```

## 出力

選べる区間の最大数を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
3
1 2
2 3
1 3
```

出力:

```
2
```


### 例 2

入力:

```
1
0 5
```

出力:

```
1
```
