# ベクトルの内積

難易度: 1 / 4 ・ トピック: matrix

長さ N の 2 つの整数ベクトル a, b が与えられる。内積を出力せよ。

## 制約

- 1 <= N <= 8
- 各要素の絶対値 <= 10^6

## 入力

```
N
a_1 ... a_N
b_1 ... b_N
```

## 出力

内積を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
2
1 2
3 4
```

出力:

```
11
```


### 例 2

入力:

```
3
-960 -804 964
157 -634 -703
```

出力:

```
-318676
```
