# アナグラム判定

難易度: 1 / 4 ・ トピック: string

2 つの文字列 S, T が与えられる。並べ替えで一致する (アナグラムである) なら `Yes`、そうでなければ `No` を出力せよ。

## 制約

- 1 <= |S|, |T| <= 10

## 入力

```
S
T
```

## 出力

`Yes` または `No` を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
listen
silent
```

出力:

```
Yes
```


### 例 2

入力:

```
abc
abd
```

出力:

```
No
```
