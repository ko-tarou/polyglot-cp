# べき乗の剰余 A^B mod M

難易度: 2 / 4 ・ トピック: math

整数 A, B, M が与えられる。A^B を M で割った余りを出力せよ。

## 制約

- 0 <= A <= 10^9
- 0 <= B <= 1000
- 1 <= M <= 10^9

## 入力

```
A B M
```

## 出力

A^B mod M を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
2 10 1000
```

出力:

```
24
```


### 例 2

入力:

```
0 0 7
```

出力:

```
1
```
