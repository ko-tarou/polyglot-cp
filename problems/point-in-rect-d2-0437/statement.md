# 点が長方形の内部か

難易度: 2 / 4 ・ トピック: geometry

軸平行な長方形の左下 (x1, y1) と右上 (x2, y2)、および点 (px, py) が与えられる。点が長方形の内部または境界上にあれば `Yes`、なければ `No` を出力せよ。

## 制約

- 座標の絶対値 <= 10^9, x1<x2, y1<y2

## 入力

```
x1 y1 x2 y2
px py
```

## 出力

`Yes` または `No` を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
0 0 10 10
5 5
```

出力:

```
Yes
```


### 例 2

入力:

```
0 0 10 10
20 5
```

出力:

```
No
```
