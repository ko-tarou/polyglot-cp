# 0/1 ナップサック

難易度: 3 / 4 ・ トピック: dp

N 個の品物 (重さ w_i, 価値 v_i) と容量 W が与えられる。重さの合計が W 以下となるよう品物を選んだときの価値の最大値を出力せよ。

## 制約

- 1 <= N <= 100
- 1 <= W <= 10000
- 1 <= w_i, v_i <= 1000

## 入力

```
N W
w_1 v_1
...
w_N v_N
```

## 出力

価値の最大値を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
3 4
2 3
1 2
3 4
```

出力:

```
6
```


### 例 2

入力:

```
1 1
2 5
```

出力:

```
0
```
