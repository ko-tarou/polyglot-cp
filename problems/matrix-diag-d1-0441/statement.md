# 正方行列の対角成分の和

難易度: 1 / 4 ・ トピック: matrix

N x N の整数行列が与えられる。主対角成分の和を出力せよ。

## 制約

- 1 <= N <= 10
- 各要素の絶対値 <= 10^6

## 入力

```
N
行列 (N 行 N 列)
```

## 出力

対角成分の和を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
2
1 2
3 4
```

出力:

```
5
```


### 例 2

入力:

```
3
112 -955 -575
-43 611 265
-585 444 229
```

出力:

```
952
```
