# うるう年判定

難易度: 1 / 4 ・ トピック: simulation

西暦 Y が与えられる。うるう年なら `Yes`、そうでなければ `No` を出力せよ。 (4 で割り切れ、かつ 100 で割り切れない、または 400 で割り切れる年がうるう年)

## 制約

- 1 <= Y <= 10^9

## 入力

```
Y
```

## 出力

`Yes` または `No` を 1 行で出力する。


## 入出力例


### 例 1

入力:

```
2000
```

出力:

```
Yes
```


### 例 2

入力:

```
1900
```

出力:

```
No
```
