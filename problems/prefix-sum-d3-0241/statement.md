# 区間和クエリ

難易度: 3 / 4 ・ トピック: array

長さ N の配列と Q 個のクエリが与えられる。各クエリ l r について a_l + a_{l+1} + ... + a_r を出力せよ (1-indexed)。

## 制約

- 1 <= N, Q <= 100000
- 各要素の絶対値 <= 10^6

## 入力

```
N Q
a_1 ... a_N
l_1 r_1
...
l_Q r_Q
```

## 出力

各クエリの答えを 1 行ずつ出力する。


## 入出力例


### 例 1

入力:

```
3 1
1 2 3
1 3
```

出力:

```
6
```


### 例 2

入力:

```
1 1
5
1 1
```

出力:

```
5
```
